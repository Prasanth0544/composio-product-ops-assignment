🔍 Executive Findings

⚡ MCP Momentum Is Accelerating

14 platforms now offer official hosted MCP servers — including Salesforce, Shopify, GitHub, Twilio, and Higgsfield — enabling direct plug-and-play agent connectivity. An additional 4 community-maintained servers extend coverage further.

🔑 OAuth 2.0 Dominates Auth

OAuth 2.0 is the primary auth standard across CRM, messaging, and ads platforms, with 64 apps requiring it. Another 82 use simple API key or token-based access — enabling fast scripting with minimal setup.

⚡ Majority Are Immediately Buildable

80 apps are directly buildable with Composio (Easy win or Buildable). Only 6 require enterprise partnerships or lack a public API entirely.

📊 High Research Confidence

80 apps carry High confidence — evidence links verified against official developer portals, API references, or SDKs. 20 apps are Medium confidence, typically requiring outreach or additional investigation. Zero apps remain at Low confidence.

The research_agent.py CLI automates link validation (multithreaded ping across all 100 docs URLs), field updates, and DuckDuckGo search loops to enrich evidence. The verify_results.py script cross-checks a 20-app random sample, computes accuracy metrics, and logs all corrections.
