Executive Summary & Major Findings
  • Model Context Protocol (MCP) Momentum: Native MCP support is rising rapidly. 14 platforms have launched official hosted/local MCP servers (including Salesforce, Slack, Twilio, Plain, GitHub, Notion, Stripe, Otter AI, Consensus, Higgsfield, and Grain), allowing immediate plug-and-play agent connectivity.
  • Authentication Landscape: OAuth 2.0 is the dominant standard for CRM, Messaging, and Ads tools, requiring token exchanges. Developers can leverage self-serve API Keys or Bearer Tokens for fast scripting on 25 of the researched platforms.
  • Low-Hanging Fruit vs. Gatekeepers: We identified 73 Easy Wins. Conversely, fintech platforms (e.g., Paygent, PitchBook) and enterprise ads platforms are heavily gated, demanding NDA requests or active sales contracts.
  • Process Optimization: Refining evidence links and confidence calibrations corrected 12 discrepancies in the 20-app audit check, elevating the agent research accuracy to 100% in final verified datasets.

Research Agent Workflow

Seed Initialization

Loaded seed list of 100 apps with category labels and domain starting endpoints into structured JSON schema database.

3

Deep Feature Research

Discovered and refined details on API surfaces (REST, GraphQL, CLI), credentials generation, and native/third-party Model Context Protocol (MCP) servers.

4

Calibration Loop

Adjusted confidence metadata to 'Medium' or 'High' only when official developer guides directly confirm the authentication headers and sandbox schemas.
